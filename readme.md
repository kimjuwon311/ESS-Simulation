## 프로그램 설명
1. 메인 시뮬레이션 프로그램(main.py)
* 매 개체 수, 비둘기 개체 수 입력
* 매와 비둘기 각각 에이전트 생성**매: H1, H2, H3... 비둘기: D1, D2, D3 ...**
* 시뮬레이션 실행 시, 비둘기와 매 무작위로 만남
	- **비둘기와 비둘기의 만남**: 서로 째려보다가 눈빛으로 상대의 기선을 제압??하는 사람이 승자**장기적이고 의식적인 싸움**. 승자는 50점 얻고, 승자와 패자 모두 시간 낭비로 추가로 -10점 얻음
	- **비둘기와 매의 만남**: 무조건 매가 이김. 승자(매)는 50점 얻고, 패자(비둘기)는 0점을 얻음
	- **매와 매의 만남**: 서로 뒤질 때까지 싸움. 승자는 50점 얻고, 패자는 중상 입으므로 -100점 얻음

	이 때, 한 번 다른 상대와 만난 개체는 싸움을 중단함.

* 시뮬레이션에서 각각의 개체군의 평균 득점을 계산 함.


* 점수표
| **항목**      | 승자  | 패자 | 중상자 | 장기전에 따른 시간낭비 |
| --------- | ----- | ---- | ------ | ---------------------- |
| **획득 점수** | +50점 | +0점 | -100점 | -10점                  |

* 프로그램과 별개의 이론: 개체군의 평균 득점이 상대적으로 높을 수록 더 이득을 본 상황이고, 더 이득을 많이 본 개체군이 자연선택 됨.

2. 도식화 프로그램(plot.py)
* 매 개체 수, 비둘기 개체 수 그리고 각 개체군의 평균 점수를 시뮬레이션 별로 입력해야 함**코드 안 리스트를 직접 수정해야 함.**
* 시뮬레이션 이름**sim1, sim2, ...**을 입력하면 x축을 'simulation' y축을 '평균 득점'으로 하여 각각 매와 비둘기의 평균 득점을 표시해 줌.
